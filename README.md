# Packet Analysis Tool for Network Security

This project is a Python-based packet analysis tool developed to capture and analyze network traffic for basic security monitoring.

## Features
- Captures live network packets
- Extracts source IP, destination IP, and protocol details
- Helps identify unusual traffic patterns

## Technologies Used
- Python
- Networking Concepts
- Cybersecurity Basics

## Learning Outcomes
- Understanding of packet structures
- Hands-on experience with network traffic analysis
- Practical exposure to cybersecurity fundamentals
